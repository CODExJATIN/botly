# Getting Started with Botly

Welcome to **Botly**, your plug-and-play chatbot solution for modern React applications.

<Divider />

## What is Botly?

Botly is a developer-first chatbot SDK that allows you to:

- Integrate conversational AI in minutes
- Use a fully-styled React component (`<BotlyBot />`)
- Or send direct API messages from your custom frontend

<Callout type="info" title="Works with All Frameworks">
Supports **Vite**, **Create React App**, **Next.js**, and other React environments.
</Callout>

<Divider />

## Create Your First Bot

To get started:

1. Log into your **Botly Dashboard**.
2. Click **Create Bot** and fill in:
   - Bot Name
   - Welcome Message
   - Optional FAQs or Context
3. Once saved, copy the **API Key** linked to this bot.

<Callout type="warning" title="Keep It Safe">
Your API key identifies your bot instance. Never expose it in public code or browser-visible config.
</Callout>

<Divider />

## Prerequisites

Make sure you have:

- A working React project (Vite, CRA, or Next.js)
- Your Botly API key
- Some familiarity with JSX and React components
