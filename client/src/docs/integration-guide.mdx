# Integration Guide

There are two integration paths:

1. Use the out-of-the-box React component  
2. Connect manually with Botlyâ€™s API

<Divider />

## 1. React Component (Fastest Way)

### Install the Package

<CodeBlock language="bash">
{`npm install botly-bots`}
</CodeBlock>

### Example Code

<CodeBlock language="jsx">
{`import './App.css';
import BotlyBot from 'botly-bots';
import 'botly-bot/dist/botly-bots.css';

function App() {
  return (
    <BotlyBot 
      apiKey={import.meta.env.VITE_BOT_API_KEY}
      initialMessage="Hello! I'm CODExJATIN, your AI assistant. How can I help you today?"
      title="CODExJATIN Assistant"
    />
  );
}

export default App;`}
</CodeBlock>

### Styling

The BotlyBot component comes styled out of the box. To override the appearance:

- Edit `botly-bots.css`
- Or wrap `<BotlyBot />` in your layout and use your own styles

<Callout type="info" title="Tip">
You can also use Tailwind, Chakra UI, or any CSS-in-JS solution to customize the wrapper.
</Callout>

## 2. Using Botly's function (botlyapi package)

Use this method if you want to make API calls from you NodeJS application directly using Botly's in built function.

### Install the Package

<CodeBlock language="bash">
{`npm install botlyapi`}
</CodeBlock>

### Sample API Call

<CodeBlock language="js">
{`  Botly("USER INPUT", "YOUR_BOTLY_API_KEY")
    .then(response => {
      console.log("Botly response:", response);
    })
    .catch(error => {
      console.error("Error calling Botly:", error);
    });`}
</CodeBlock>
